Glyphicons

This section illustrate how to add Glyphicons Icon Picker to Elements withs options. Very first you need to include Icon fonts css glyphicons.css in page header and tell this plugin to add only Glyphicons icons as Icon Picker.

html head 
<link href="css/glyphicons.css" rel="stylesheet" type="text/css" />

In input element

Configure Icon Picker to a HTML input elemnt on focus.

HTML 
<input id="example1" type="text" class="form-control" placeholder="icon class" >
JavaScript 
$(function () {
      $("#example1").iconPicker({'icons':'glyphicons'});
  });
OUTPUT

As input element

Configure Icon Picker to add Icons to a HTML input elemnt on focus.

HTML 
<div class="input-group">
  <input type="text" class="form-control" placeholder="icon class" aria-describedby="basic-demo1">
  <span class="input-group-addon" id="basic-demo1"></span>
</div>
JS 
<script type="text/javascript">
  $(function () {
      $("#basic-demo1").iconPicker('glyphicons');
  });
</script>
